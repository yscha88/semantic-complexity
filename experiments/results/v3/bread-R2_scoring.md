## R1+R2 통합 결론

### 2레포 × 2모델에서 일관되게 관찰된 것

1. **SKILL(C)은 탐지 범위를 좁힌다** — 체크리스트가 "완료 조건"을 만들어 탐색 조기 종료
2. **SKILL+자율탐색(D)은 C의 시야 축소를 해소하면서 구조화를 유지한다** — A 이상의 TP + C 수준의 구조화
3. **이 패턴은 모델(GPT-5.4, Sonnet-4.6)과 코드(R1 93줄, R2 217줄)에 무관하게 재현된다**
4. **SKILL은 양방향 주의 제어 도구다** — 체크리스트로 범위 축소(정밀도↑), 자율 탐색으로 범위 확장(탐지↑)

### Phase α 성공 기준 재판정

> "Group C의 정밀도가 Group A보다 높음" → **혼합** (R2 GPT: C 0.83 < A 1.00, R2 Son: C 1.00 = A 0.86)
> "Group C의 구체성이 Group A보다 높음" → **구조화에서 예** (C: 2 > A: 1, 양 모델 양 레포)
> "Group C가 A에서 놓친 실제 이슈를 탐지" → **예** (AUTH_FLOW, UnicodeDecodeError 일부)

**당초 성공 기준(A<C)은 단순 TP 수로는 실패했지만, D(SKILL+자율탐색)를 발견하여 "SKILL이 주의 방향을 제어한다"는 더 유의미한 결론에 도달했다.**
